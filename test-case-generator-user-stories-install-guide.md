# Install Guide: Test Case Generator from User Stories

> Automatically generate comprehensive test cases from user stories and acceptance criteria using BDD patterns, equivalence partitioning, and risk-based prioritization

**Version:** 1.0.0

---

## Quick Install (Recommended)

```bash
npx @qaskills/cli add test-case-generator-user-stories
```

This auto-detects your AI agent and installs the skill in the correct location.
